# AVO.AI

A modern web application for finding PhD opportunities, powered by AI.

## Features

- **AI-Powered Search**: Smart filtering for academic positions.
- **Modern Interface**: React-based UI with glassmorphism effects.
- **Interactive**: 3D background elements.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Three.js
- **Backend**: Python (Flask)

## Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/iamr7d/AVO.AI.git
   cd AVO.AI
   ```

2. **Install Dependencies**

   - Backend: `pip install -r requirements.txt` (in `backend/` or root)
   - Frontend: `npm install` (in `frontend/`)

3. **Run**
   - Start Flask API.
   - Start React App (`npm start`).
